# Analise-Notas


Projeto simples em Python que analisa uma lista de notas e separa em duas categorias:
- Notas acima ou iguais à média
- Notas abaixo da média


Este projeto é voltado para iniciantes que estão aprendendo sobre **listas** e **funções** em Python.


---


## Funcionalidades


- Recebe uma lista de notas como entrada
- Calcula a média das notas
- Cria duas listas: notas acima/igual à média e notas abaixo da média
- Retorna as duas listas para o usuário
